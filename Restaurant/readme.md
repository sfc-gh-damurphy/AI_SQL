# Restaurant Review Lab

This section of the lab focuses on **restaurant review analysis** and includes the following files:

* **`AISQL Restaurant Review.ipynb`**: The main Jupyter Notebook for the lab, containing the core code and explanations.
* **`snowbooks_extras.py`**: A helper Python file that provides additional functions and utilities imported by the notebook.
* **`doordash_100.csv`**: A dataset containing DoorDash reviews, used for analysis within the lab.
* **`Food Images/`**: A directory containing images of food, intended for analysis by a Large Language Model (LLM).
